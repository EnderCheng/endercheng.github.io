  The pervasiveness of location-based mobile devices has greatly promoted the development of the location-based service. The polygons spatial query in outsourced cloud which is one of the major technologies of the location-based service has attracted considerable interests recently. However, the polygons spatial query still faces many challenges including spatial information security and privacy preservation. In this paper,we proposed an efficient and privacy-preserving polygons spatial query framework for location-based services, called Polaris. With Polaris, the data owner first outsourced the encrypted spatial information to cloud, and then the registered user can query any convex range (not confined to the circular or rectangle traditionally) and get accurate results without divulging his/her query information to the data owner and cloud. Specifically, an improved homomorphic encryption technology over composite order group is employed to construct a special polygon spatial (convex) query algorithm (SPSQ), which achieves the privacy preservation of users query information, the security of the LBS data outsourced to cloud and the accuracy of spatial range query results. Detailed secure analysis shows that our proposed Polaris can resist various security threats. In addition, Polaris framework is implemented over a smart phone, two workstations with real spatial dataset, and the simulation results demonstrate that the proposed Polaris framework is highly efficient at smart phone and can be implemented effectively in the cloud server.
